#### About

A boilerplate for developing web applications. No servers. No compiler platforms.

#### Included

- Bootstrap
- jQuery
- Lodash
- Moment
- Sortable
- Chart JS
- Tabulator

#### Run

1. Open `index.html` in a browser.

#### Use

1. Just start adding your code.
	- `index.html`
	- `styles.css`
	- `scripts.js`
